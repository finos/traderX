# Spec Kit Component: ingress

## Responsibilities

- Provide consolidated edge routing for browser/API access in containerized states.
- Terminate external host routing toward UI and backend services.

## Covered Flows

- Not part of baseline uncontainerized startup flow.
- Used in containerized and hosted topologies.
