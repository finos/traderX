# Quickstart: PostgreSQL Database Replacement

## 1) Generate State 009

```bash
bash pipeline/generate-state.sh 009-postgres-database-replacement
```

## 2) Start / Verify / Test / Stop

```bash
./scripts/start-state-009-postgres-database-replacement-generated.sh
./scripts/status-state-009-postgres-database-replacement-generated.sh
./scripts/test-state-009-postgres-database-replacement.sh
./scripts/stop-state-009-postgres-database-replacement-generated.sh
```
