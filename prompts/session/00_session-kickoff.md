# TraderX - Agentic Session Kickoff

## Purpose

Coordinate an agentic refactor of TraderX into a learning-path-first, multi-state architecture with a prompt pack.

## Objectives

1. Normalize guide front-matter in `docs/guide`.
2. Map guides to `states/00..05`.
3. Keep prompt pack under `prompts/**`.
4. Keep Level 3 Solo demo under `states/03-service-mesh/solo-demo`.
5. Produce a PR-ready validation checklist.

## Tasks

1. Inventory guide files and detect front-matter gaps.
2. Ensure each state folder has `README.md` and `scripts/verify.sh`.
3. Build or refresh `docs/learning-paths/index.md`.
4. Validate prompt files are present and linked from `AGENTS.md`.
5. Run local checks and summarize findings.

## Required Output

- Patch plan
- Updated file tree
- Validation output
- Follow-up risks and open questions
