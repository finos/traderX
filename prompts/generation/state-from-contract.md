# Prompt - Generate Implementation from Contracts (Level 4)

## System

Generate or update contract-driven implementation under `states/04-contract-driven` from specs in `specs/contracts/*`.

## Requirements

1. Generate/update services, DTOs, validation, and contract tests.
2. Add build/run/verify instructions.
3. Keep Levels 0-3 runnable and non-breaking.
4. Prefer additive namespaces/modules when conflicts appear.

## Output

- File tree of changes
- Diffs or stubs for generated files
- Build/start/verify commands
- Backward-compatibility notes
- Open questions about contract ambiguities
