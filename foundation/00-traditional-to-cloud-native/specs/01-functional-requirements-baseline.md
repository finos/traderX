# 01 Functional Requirements Baseline

These baseline FRs describe what TraderX must do independent of deployment style.

## FR-001 Account Lifecycle

System supports creating, listing, and managing accounts and account-user associations.

## FR-002 Trade Capture

System accepts trade orders with symbol, side, quantity, and account context.

## FR-003 Position Tracking

System computes and exposes positions from booked trades.

## FR-004 Reference Data Access

System provides symbol/reference lookup needed for trading workflows.

## FR-005 People Directory Integration

System exposes people metadata for account-user mapping and UI workflows.

## FR-006 UI Trading Workflow

UI supports selecting accounts, entering trades, and viewing trade/position blotters.

## FR-007 Trade Event Propagation

System propagates trade updates/events to interested components.

## FR-008 Service Health

Core services expose health/readiness endpoints for operational checks.
