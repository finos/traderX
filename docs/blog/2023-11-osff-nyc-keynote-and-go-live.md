# OSFF NYC Keynote Demo and Go-Live Momentum

Published: **November 2023**

Fall 2023 marked an important visibility moment for TraderX: the project was presented in New York at the Open Source in Finance Forum (OSFF), including keynote-demo context that helped position TraderX as a practical learning project for modern trading-system architecture.

Around the same period, repository changes reflected go-live readiness:

- Documentation and branding updates were prepared for public visibility (`#72`).
- Ongoing dependency and security updates continued across languages.
- The project identity became more intentional, including demo discoverability.

Relevant event reference:

- [Open Source in Finance Forum New York 2023](https://events.linuxfoundation.org/archive/2023/open-source-finance-forum-new-york/)

This milestone mattered beyond publicity. It reinforced the need for TraderX to serve two audiences at once:

- newcomers looking for a runnable, understandable system,
- maintainers and contributors pushing for modernization and extensibility.

Balancing those audiences remains central to the project's evolution.
