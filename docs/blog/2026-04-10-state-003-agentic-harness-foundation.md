# State 003 - Agentic Harness Foundation

State `003-agentic-harness-foundation` was added as an early, non-domain feature state so generated codebases are easier to work with using agentic tooling before larger architecture and runtime transitions.

## What This State Adds

- baseline agent-oriented repository guidance in generated outputs
- normalized non-code metadata scaffolding for codebase understanding
- contribution guidance that keeps generated repos focused on local experimentation while directing upstream enhancements back to spec/source states

## Why It Sits Before C0

Placing this state before the first convergence baseline means all later generated states inherit a consistent foundation for:

- onboarding with coding agents
- predictable repository documentation
- contribution path clarity

This keeps downstream generated branches more usable without mixing these concerns into later feature-specific states.
