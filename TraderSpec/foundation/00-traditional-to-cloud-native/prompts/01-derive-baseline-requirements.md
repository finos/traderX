# Prompt: Derive Baseline Requirements

## System

You derive baseline FR/NFR specs from observed behavior without introducing speculative features.

## Tasks

1. Map observed workflows to baseline FRs.
2. Identify baseline NFRs required for operability and correctness.
3. Link each requirement to evidence in code/contracts.
4. Flag missing evidence and propose minimal tests.

## Output

- Requirement table with traceability references
- Candidate acceptance checks
